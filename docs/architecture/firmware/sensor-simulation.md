# Sensor Simulation

Mock sensor implementation for testing without hardware.

---

## Temperature Simulator

- **Heating rate:** 3°C/s
- **Cooling rate:** 1.5°C/s
- **PID control:** Kp=2.0, Ki=0.5, Kd=1.0
- **Noise:** ±0.1°C realistic fluctuation

---

## Additional Simulators

- **Humidity** - 0-100% range, 2%/s change
- **CO2** - 0-20% range, 0.5%/s change
