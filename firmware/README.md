# Axionyx ESP32 Firmware

Demo firmware for ESP32-WROOM-32 biotech IoT devices.

---

## Quick Start

```bash
# Navigate to device directory
cd dummy/dummy_demo

# Build and upload
pio run --target upload

# Monitor serial output
pio device monitor
```
